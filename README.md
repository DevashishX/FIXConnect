# FIXConnect

FIXConnect is a lightweight toolkit for building workflows around the **FIX (Financial Information eXchange)** protocol.

The goal of this repository is to provide a small set of scripts and helpers that make it easier to:
- format and send FIX-related requests
- trigger common order operations (e.g., **modify** and **cancel**)
- provide a simple local UI (Tkinter) for manual/interactive actions

## Uses
- **Trading ops / tooling**: quickly submit cancel/modify requests during testing or in controlled environments.
- **FIX workflow prototyping**: serve as a starting point for building a more complete FIX client or adapter.
- **Internal automation**: integrate the scripts into cron jobs or pipelines to automate repetitive request flows.
- **Educational reference**: learn how small utilities can be organized around FIX-style request/response workflows.

## Technologies
- **Python** (primary): scripts and helpers.
  - **Tkinter/ttk**: simple GUI windows (generated by PAGE) for interactive flows.
  - **Requests**: HTTP client for making requests.
  - **Flask** + **Flask-Cors**: lightweight web service / API support (see `requirements.txt`).
- **Tcl/Tk** (`.tcl` files): UI definitions generated/used alongside the Python GUI code.

## What’s in this repo
- `cancel.py`, `modify.py`: GUI entrypoints for cancel/modify flows.
- `*_support.py`: support modules used by the generated GUI.
- `requestformat.py`, `requestlib.py`: small helpers for request formatting / utilities.
- `requirements.txt`: Python dependencies.

## Setup
```bash
python -m venv .venv
source .venv/bin/activate  # on Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Run
```bash
python cancel.py
# or
python modify.py
```
